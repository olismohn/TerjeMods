## Community Ownership and Freedom of Use

TerjeMods are released for the benefit of the entire DayZ community. The code, configurations, and related materials are intended to be freely available for study, modification, and use by players, server owners, and developers. Everyone is welcome to explore how the modifications work, adapt them to their own projects, create derivative works, or integrate parts of the code into other non-commercial community initiatives.

The project is distributed with the understanding that DayZ is driven by its community, and these modifications exist thanks to the passion and support of that community. This repository is provided as a contribution to that shared environment.
